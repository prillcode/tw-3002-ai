# Keyboard Reference

## Global

| Key | Action |
|-----|--------|
| **↑ / ↓** | Navigate menus |
| **Enter** | Select / Confirm |
| **Esc** | Go back |
| **Q** | Quit (with confirmation) |

## Sector Screen

| Key | Action |
|-----|--------|
| **↑ / ↓** | Select warp destination |
| **Enter** | Jump to selected sector |
| **M** | Open Market (if port present) |
| **D** | Dock at StarDock (if present) |
| **N** | Open Navigation Log |
| **H** | Toggle Help screen |
| **L** | Leaderboard |
| **Esc** | Return to main menu |

## Market Screen

| Key | Action |
|-----|--------|
| **↑ / ↓** | Select commodity |
| **B** | Buy mode |
| **S** | Sell mode |
| **Enter** | Confirm trade |
| **Esc** | Back to sector |

## Combat Screen

| Key | Action |
|-----|--------|
| **↑ / ↓** | Select action |
| **Enter** | Confirm |
