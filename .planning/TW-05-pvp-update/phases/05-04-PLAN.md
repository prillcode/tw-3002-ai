## Objective

## Context

- Leaderboard API already exists (`/api/leaderboard`)

## API Endpoints

### Existing: `GET /api/leaderboard?galaxyId=&limit=`
Already returns net worth rankings. Extend with kill/death data.

### New: `GET /api/leaderboard/kills?galaxyId=&limit=`
Top killers.

**Response:** `{ leaderboard: [{ rank, shipName, playerName, kills, kdr }] }`

### New: `GET /api/leaderboard/deaths?galaxyId=&limit=`
Most deaths (fun alternative leaderboard).

### New: `GET /api/leaderboard/wanted?galaxyId=&limit=`
Most wanted players.

### Existing: `GET /api/player/stats?galaxyId=`
Return full stats for authenticated player.

**Response:**
```json
{
  "kills": 12,
  "deaths": 3,
  "kdr": 4.0,
  "netWorth": 450000,
  "wanted": false,
  "wantedKillCount": 0,
  "rank": "Private"
}
```
