# 05-05 — PvP Polish & Balance

## Objective
PvP feels fair, tense, and fun — not frustrating or exploitable.

## Context
- See `../BRIEF.md`
- See `../ROADMAP.md`
- Core systems are built — now tune them

## Tasks
1. Combat balance pass:
   - Test maxed-out ship vs new ship — ensure newbies aren't one-shot
   - Add level scaling: damage reduced by 50% if target is < 50% of attacker's net worth
2. Escape mechanics:
   - Engine upgrades add +5% flee chance per tier
   - Base flee: 25%, max: 50%
3. Cloaking device (upgrade):
   - Hide from player scanners for 1 turn
   - Costs 5,000 cr at StarDock
4. Ship insurance (StarDock):
   - Pay 5% of net worth to reduce death penalty from 10% to 5%
   - One-time purchase, lasts until next death
5. Pacifist mode:
   - Player can opt out of PvP (settings)
   - Gets a 🕊️ badge, no PvP rewards, can't attack others
   - Can't be attacked (hard opt-out)
6. Admin tools:
   - `POST /admin/pvp/reset-wanted` — clear wanted status
   - `GET /admin/pvp/kill-log` — inspect recent kills
   - Ban griefers by player ID

## Verification
- [ ] Newbie survives 3 hits from maxed ship
- [ ] Flee chance increases with engine upgrades
- [ ] Insurance reduces death penalty
- [ ] Pacifist players can't be attacked
- [ ] Admin can reset wanted status

## Done When
- PvP is balanced across ship levels
- Risk/reward feels meaningful
- Protections are sufficient but not oppressive
- Admin has tools to manage abuse

## Notes
- Insurance is the "soft protection" — players pay for safety
- Pacifist mode is the "hard protection" — complete opt-out
- The goal is tension, not cruelty. Players should fear PvP, not avoid the game because of it
