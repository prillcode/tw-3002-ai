# PLAN: TW-01 Phase 3 — Welcome Screen Polish

**Work Item:** TW-01-cli-client  
**Phase:** 3 of 7  
**Estimated Duration:** 3-4 hours  
**Goal:** Authentic BBS-era welcome screen with ANSI art, refined styling, and ship name prompt

---

## Objective
Transform the functional but plain WelcomeScreen into an authentic BBS-era experience. Add ANSI art title, proper color scheme, animated "press any key" prompt, and a ship naming flow that feels like logging into a 90s door game.

---

## Context

### What We Have (Phase 2)
- WelcomeScreen with rounded border box
- Basic menu (New Game, Quit)
- Functional but visually plain
- No ship naming flow yet

### What We're Building (Phase 3)
- ANSI art title (ASCII-style "TW 3002 AI")
- Authentic BBS color scheme (cyan/magenta/yellow on black)
- "Press any key to continue" prompt
- Ship name input screen
- Visual polish that screams "1994 BBS door game"

---

### Task 5: Authentic BBS Color Scheme (30 min)
**Apply classic BBS colors throughout**

**Classic BBS Palette:**
- Background: Black (terminal default)
- Primary: Bright Cyan (titles, highlights)
- Secondary: Bright Magenta (accents)
- Info: Bright Yellow (important text)
- Muted: Dark Gray (secondary text)
- Success: Bright Green (confirmations)
- Danger: Bright Red (errors/warnings)

**Update WelcomeScreen:**
- Title: Cyan on black
- Menu: White text, cyan highlight for selected
- Borders: Cyan single lines
- Press any key: Yellow blinking

**Verification:**
- [ ] Colors match BBS aesthetic
- [ ] Readable on black background
- [ ] Consistent across all states
