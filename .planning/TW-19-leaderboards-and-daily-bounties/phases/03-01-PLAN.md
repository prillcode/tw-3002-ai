## Tasks

### Task 1: Add new leaderboard API sorts
**File:** `cloud/src/routes/news.ts` — `handleLeaderboard()`

Add two new `sort` options:

#### `sort=planets`
```sql
SELECT ps.ship_name, ps.class_id, ps.net_worth, ps.kills, ps.deaths,
       ps.alignment, ps.experience, ps.rank, ps.commissioned,
       p.display_name, p.email,
       (SELECT COUNT(*) FROM planets pl WHERE pl.owner_id = ps.player_id AND pl.galaxy_id = ps.galaxy_id) as planets_held
FROM player_ships ps
JOIN players p ON ps.player_id = p.id
WHERE ps.galaxy_id = ?
ORDER BY planets_held DESC
LIMIT ?
```

#### `sort=experience`
```sql
SELECT ps.ship_name, ps.class_id, ps.net_worth, ps.kills, ps.deaths,
       ps.alignment, ps.experience, ps.rank, ps.commissioned,
       p.display_name, p.email
FROM player_ships ps
JOIN players p ON ps.player_id = p.id
WHERE ps.galaxy_id = ?
ORDER BY ps.experience DESC
LIMIT ?
```

Also update the default query (`sort=networth`) to include `alignment`, `experience`, `rank`, `commissioned` so the profile modal has full data.
