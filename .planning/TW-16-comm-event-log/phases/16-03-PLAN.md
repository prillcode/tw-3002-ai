# 16-03 — Event Log UI

## Objective
Display the event log in the game client with a full view and a mini ticker on the sector screen.

## Context
- See `../BRIEF.md` and `../ROADMAP.md`
- Phases 1–2 created the store and wired all actions to generate entries
- Now we need to make them visible

## Tasks

### 1. Extend NavigationView → ActivityLogView
The existing NavigationView only shows visited sectors. Transform it into a unified activity log:

- Rename conceptually (keep file or create new `ActivityLogView.vue`)
- Two tabs at top: **Activity Log** | **Sector Visits**
- **Activity Log tab** shows categorized events from `eventLog` store:
  - Each entry: `<icon> <tag> <timestamp> <text>`
  - Color-coded by tag (cyan NAV, yellow TRD, red CMB, magenta UPG, white EVT, muted SYS)
  - Timestamp formatted as `HH:MM:SS` (session-local, no date needed)
  - Auto-scroll to bottom, scrollback to see older entries
  - Last 20 visible without scrolling, 100 in store
- **Sector Visits tab** preserves current NavigationView behavior (sector list with danger dots)
- Keyboard: `Esc` to go back, `Tab` to switch between tabs

### 2. Mini event ticker on SectorView
Add a compact event ticker above the warp lanes in SectorView:

- Shows last 3 events as single-line entries
- Each: colored icon + truncated text (max 60 chars)
- Click/tap on ticker opens full ActivityLogView
- Auto-updates as new events arrive (reactive)
- Visual style: subtle, doesn't dominate the sector view

```html
<!-- Example -->
<div class="event-ticker space-y-0.5 mb-3 text-xs font-mono">
  <div class="text-terminal-red">⚔ Victory over Pirate Raider +340 cr</div>
  <div class="text-terminal-cyan">🧭 Warped to Sector 42</div>
  <div class="text-terminal-yellow">💰 Sold 50 ore for 12,500 cr</div>
</div>
```

### 3. Update router
- Add route for `/galaxy/:id/log` → ActivityLogView (or reuse `/galaxy/:id/nav`)
- Wire keyboard shortcut in SectorView (e.g., `L` for log, keep `N` for sector visits)

### Files to modify
- `web/game/src/views/NavigationView.vue` — extend with activity log
- `web/game/src/views/SectorView.vue` — add ticker, wire `L` key
- `web/game/src/router.ts` — add route if new view
