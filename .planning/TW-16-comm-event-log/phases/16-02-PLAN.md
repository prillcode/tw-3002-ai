# 16-02 — Wire All Actions to Event Log

## Objective
Replace every `ship.message = ...` assignment and every place the game produces a user-visible result with `eventLog.addEvent()` calls.

## Context
- See `../BRIEF.md` and `../ROADMAP.md`
- Phase 1 created the store and types; `ship.message` is removed
- Now every view that produces output needs to emit typed events instead

## Tasks

### Navigation events (NAV)
- **SectorView.vue** — After successful warp:
  - `addEvent('NAV', 'Warped to Sector 42', 42)`
- **SectorView.vue** — After fighter encounter resolution (retreat):
  - `addEvent('NAV', 'Retreated from Sector 43 fighters', 43)`

### Trade events (TRD)
- **MarketView.vue** — After buy:
  - `addEvent('TRD', 'Bought 50 ore for 5,000 cr', sector)`
- **MarketView.vue** — After sell:
  - `addEvent('TRD', 'Sold 30 organics for 4,200 cr', sector)`

### Combat events (CMB)
- **CombatView.vue** — On victory:
  - `addEvent('CMB', 'Victory over Pirate Raider +340 cr', sector)`
- **CombatView.vue** — On flee:
  - `addEvent('CMB', 'Escaped from Pirate Raider', sector)`
- **CombatView.vue** — On bribe:
  - `addEvent('CMB', 'Bribed Pirate Raider for 500 cr', sector)`
- **CombatView.vue** — On defeat/destruction:
  - `addEvent('CMB', 'Destroyed by Pirate Raider in Sector 42', sector)`
- **CombatView.vue** — On fighter encounter resolved:
  - `addEvent('CMB', 'Cleared 25 fighters in Sector 43 (Offensive)', sector)`

### Upgrade events (UPG)
- **StarDockView.vue** — After upgrade purchase:
  - `addEvent('UPG', 'Purchased Hull Plating Mk II', sector)`
- **StarDockView.vue** — After insurance purchase:
  - `addEvent('UPG', 'Insurance activated (7 days)', sector)`

### General events (EVT)
- **SectorView.vue** — After fighter deploy:
  - `addEvent('EVT', 'Deployed 50 fighters (Defensive) in Sector 42', sector)`
- **SectorView.vue** — After fighter recall:
  - `addEvent('EVT', 'Recalled 50 fighters from Sector 42', sector)`
- **SectorView.vue** — After fighter purchase:
  - `addEvent('EVT', 'Bought 100 fighters at StarDock', sector)`
- **SectorView.vue** — News ticker items (top story only):
  - `addEvent('EVT', newsHeadline)`
- **LoginView.vue** — Digest death events:
  - `addEvent('EVT', 'While away: destroyed 1 time(s)')`

### System events (SYS)
- **Ship store** — Turn regeneration on load:
  - `addEvent('SYS', '+15 turns regenerated while away')`
- **Ship store** — On error:
  - `addEvent('SYS', 'Error: Failed to load ship')`

### Files to modify
- `web/game/src/views/SectorView.vue`
- `web/game/src/views/MarketView.vue`
- `web/game/src/views/CombatView.vue`
- `web/game/src/views/StarDockView.vue`
- `web/game/src/views/LoginView.vue`
- `web/game/src/stores/ship.ts`
