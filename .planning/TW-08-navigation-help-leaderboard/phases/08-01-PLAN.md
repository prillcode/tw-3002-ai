# 08-01 — Navigation Log in Cloud Mode

## Objective
Allow players to view their flight history in cloud mode by pressing `N`.

## Context
- See `../BRIEF.md`
- See `../ROADMAP.md`
- `NavigationScreen` already exists for local mode
- Cloud mode has no visited-sector tracking yet

## Tasks
1. Add `visitedSectorIds` state to CloudSectorScreen
2. On every successful jump, append new sector to the list
3. On ship destruction, append `-1` as blast marker (same as local)
4. Add `onN` to `useKeyHandler` that opens NavigationScreen
5. Pass `visitedIds`, `currentSectorId`, `startingSectorId` to NavigationScreen
6. Handle `onBack` from NavigationScreen to return to sector view

## Verification
- [ ] Jump through 3 sectors
- [ ] Press `N` — see breadcrumb trail
- [ ] Get destroyed — see blast marker
- [ ] Press `N` or `Esc` to close

## Done When
- Navigation Log works in cloud mode
- Blast markers appear on death
- Breadcrumb shows last 8 hops

## Notes
- No server tracking needed — purely client-side state
- Same NavigationScreen component as local mode
