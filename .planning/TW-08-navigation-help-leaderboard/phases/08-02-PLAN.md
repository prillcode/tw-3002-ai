# 08-02 — Help Screen for Cloud Mode

## Objective
Add a cloud-specific help screen accessible via `H` key in CloudSectorScreen.

## Context
- See `../BRIEF.md`
- See `../ROADMAP.md`
- `HelpScreen` supports `sector`, `market`, `combat`, `stardock`, `general` contexts
- No `cloud` context exists

## Tasks
1. Add `cloud` to `HelpContext` union type in `HelpScreen.tsx`
2. Add cloud controls to `CONTROLS` record:
   - `↑↓` — Select warp destination
   - `Enter` — Jump to selected sector
   - `M` — Open Market (if port exists)
   - `D` — Enter StarDock (if present)
   - `N` — Navigation Log
   - `H` — Help
   - `Esc` — Back / Menu
   - `Q` — Quit
3. Wire `onH` in CloudSectorScreen to call `setAppMode('help')` (or internal help state)
4. Set `prevMode` correctly so `Esc` returns to cloud sector

## Verification
- [ ] Press `H` in cloud sector — see help screen
- [ ] Controls listed match actual keys
- [ ] Press `H` or `Esc` to close

## Done When
- Help screen accessible from cloud mode
- Cloud controls documented
- Close behavior works

## Notes
- Minimal change — just add context and wire handler
