# 15-01 — Alignment Tracking & Actions

## Objective
Track player alignment and experience, and shift alignment based on in-game actions.

- TW-05 PvP infrastructure is live (kill tracking, loot, respawn)

- Reference: `../lore-reference/alignment/evil-path.md`

## Schema Changes

### `player_ships` column additions
```sql
ALTER TABLE player_ships ADD COLUMN alignment INTEGER NOT NULL DEFAULT 0;
ALTER TABLE player_ships ADD COLUMN experience INTEGER NOT NULL DEFAULT 0;
ALTER TABLE player_ships ADD COLUMN rank INTEGER NOT NULL DEFAULT 1;
```

> Alignment range: -1000 to +1000. Experience starts at 0 and grows exponentially.

## Alignment Gain/Loss Events

### Good Actions (+alignment)
| Action | Alignment Change | Trigger |
|--------|-----------------|---------|
| Kill evil NPC/player | +amount (target-dependent) | Combat resolution |
| Post bounty | +1 per 1,000 cr spent | Bounty posting (future) |
| Build planet | +1 per 2,000 cr invested | Planet creation/advancement |
| Upgrade port | +1 per 5,000 cr invested | Port upgrade action |
| Pay taxes | +1 per 1,500 cr | Tax payment (Stardock) |

### Evil Actions (-alignment)
| Action | Alignment Change | Trigger |
|--------|-----------------|---------|
| Destroy port | -50 | Port destruction (future) |
| Kill good player | -amount (target-dependent) | PvP combat resolution |
| Rob port (on bust) | -penalty | Bust event |
| Steal cargo (on bust) | -penalty | Bust event |

## API Endpoints

### `GET /api/player/alignment?galaxyId=`
Get current alignment, experience, and rank.

**Response:**
```json
{
  "alignment": 150,
  "alignmentLabel": "good",
  "experience": 1250,
  "rank": 5,
  "rankTitle": "Sergeant",
  "nextRankExp": 2048,
  "commissioned": false,
  "canRob": false,
  "canBuyISS": false
}
```

### `POST /api/player/pay-taxes`
Pay taxes to gain alignment.

**Body:** `{ galaxyId, amount }`

**Validation:**
- Player must be in FedSpace
- Amount must be positive and player has sufficient credits
- Alignment gain = floor(amount / 1500)
- Alignment capped at +1000

**Response:** `{ success, alignmentGained, newAlignment, remainingCredits }`

### `POST /api/player/commission`
Request commission at Stardock.

**Body:** `{ galaxyId }`

**Validation:**
- Player must be at Stardock (stardock sector)
- Alignment must be ≥ 1000
- Sets alignment to exactly 1000 if below (one-time boost)

**Response:** `{ success, commissioned, alignment }`

## Notes
- **Alignment is hard to change by design.** A player who wants to switch from evil to good must invest hundreds of thousands of credits.
- **Experience is permanent.** Even if alignment shifts, experience (and thus rank) never decreases.
- **Tax payment is intentionally expensive.** 1,500 cr per alignment point means +1000 alignment costs 1.5M credits via taxes alone.
- **Cross-pod farming excluded.** We're not implementing the "hit an evil partner with 1 fig" exploit for alignment gain.
