# 10-02 — Vue Game Client Scaffold

## Objective
Get the Vue 3 SPA game client building and running locally at `web/game/`.

## Tasks
1. Install dependencies: `cd web/game && npm install`
2. Verify dev server starts: `npm run dev` (should serve on localhost:5173)
3. Test auth flow: register → ship creation → sector view
4. Test keyboard shortcuts and navigation
5. Fix any TypeScript or build issues

## Verification
- [ ] `npm run dev` starts without errors
- [ ] `localhost:5173` shows the login screen
- [ ] Registration creates a player and stores token in localStorage
- [ ] Ship creation works and redirects to sector view
- [ ] Sector view shows ship status, warp lanes, sector info
- [ ] Jumping to a neighbor sector updates the view
- [ ] `npm run build` produces a `dist/` directory

## Done When
- Vue SPA builds successfully
- Auth flow works end-to-end against the live API
- Sector view renders with real data
- No console errors
