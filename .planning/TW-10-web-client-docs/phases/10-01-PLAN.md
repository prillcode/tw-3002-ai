# 10-01 — Astro Docs Site Scaffold

## Objective
Get the Astro marketing/docs site building and running locally at `web/main/`.

## Tasks
1. Install dependencies: `cd web/main && npm install`
2. Verify dev server starts: `npm run dev` (should serve on localhost:4321)
3. Test landing page renders correctly
4. Test guide pages render correctly
5. Fix any TypeScript or build issues

## Verification
- [ ] `npm run dev` starts without errors
- [ ] `localhost:4321` shows the landing page
- [ ] `localhost:4321/guide/getting-started` shows the guide
- [ ] `npm run build` produces a `dist/` directory

## Done When
- Astro site builds successfully
- Landing page and guide pages render
- No console errors
