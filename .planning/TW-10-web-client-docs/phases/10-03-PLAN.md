# 10-03 — Cloudflare Pages Deployment

## Objective
Deploy both web properties to Cloudflare Pages with the correct custom domains.

## Tasks
1. Create Cloudflare Pages project for `web/main` → `playtradewars.net`
2. Create Cloudflare Pages project for `web/game` → `portal.playtradewars.net`
3. Configure build commands and output directories
4. Set up custom domain DNS records
5. Verify end-to-end: browser → web client → API → D1

## Verification
- [ ] `playtradewars.net` serves the Astro landing page
- [ ] `playtradewars.net/guide/getting-started` serves docs
- [ ] `portal.playtradewars.net` serves the Vue SPA
- [ ] Web client can register, create ship, and view sector
- [ ] CORS headers are correct in API responses

## Done When
- Both sites are live on their custom domains
- Web client can fully interact with the API
- No CORS errors in browser console
