## Schema Changes

### `sector_fighters` table (new)
```sql
CREATE TABLE sector_fighters (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  galaxy_id INTEGER NOT NULL,
  sector_index INTEGER NOT NULL,
  owner_id INTEGER NOT NULL,
  count INTEGER NOT NULL DEFAULT 0,
  mode TEXT NOT NULL DEFAULT 'defensive', -- defensive | offensive | tolled
  deployed_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  UNIQUE(galaxy_id, sector_index, owner_id)
);
```

### `player_ships` column addition
Add `fighters` column to track ship-carried fighters (separate from deployed):
```sql
ALTER TABLE player_ships ADD COLUMN fighters INTEGER NOT NULL DEFAULT 0;
```

> **Note:** In TW2002, starting ships carry 30 fighters. We'll seed new ships with 30.

## API Endpoints

### `POST /api/fighters/buy`
Purchase fighters at StarDock.

**Body:** `{ galaxyId, quantity }`

**Validation:**
- Player must be in a stardock sector
- Player has sufficient credits (price: 100 cr/fighter, tunable)
- Quantity > 0

**Response:** `{ success, quantity, cost, remainingCredits, shipFighters }`

### `POST /api/fighters/deploy`
Deploy carried fighters into current sector.

**Body:** `{ galaxyId, sectorId, quantity, mode }`

**Validation:**
- Player must be in target sector
- Player has sufficient carried fighters
- Mode is one of: `defensive`, `offensive`, `tolled`
- Cannot deploy in FedSpace (safe sectors)

**Response:** `{ success, quantity, mode, remainingShipFighters, sectorFighters }`

### `GET /api/fighters/sector?galaxyId=&sectorId=`
List fighters in a sector.

**Response:** `{ fighters: [{ ownerId, ownerName, count, mode, deployedAt }] }`

### `POST /api/fighters/recall`
Recall deployed fighters back to ship.

**Body:** `{ galaxyId, sectorId, quantity }`

**Validation:**
- Player must be in target sector
- Player has sufficient deployed fighters

**Response:** `{ success, quantity, shipFighters, sectorFighters }`

## Web Client Changes

### StarDockView.vue
- Add "Fighter Bay" section
- Show current carried fighters
- Buy fighters: quantity selector, price display, credit check
- Cost: 100 cr/fighter (displayed as "100 cr / fighter")

### SectorView.vue
- Add "Fighters" line to ship status panel:
  - `Fighters: {{ ship.ship.fighters }} carried / {{ deployedFighters }} deployed`
- Add fighter deployment button in Actions panel:
  - "🚀 Deploy Fighters" (always visible, disabled if 0 carried or in FedSpace)
- Sector info panel: show fighter presence indicator
  - "⚔ 3,200 hostile fighters" (if other players' fighters present)
  - "🛡 Your fighters: 500 (defensive)" (if own fighters present)

### DeployFightersModal.vue (new)
- Quantity selector (max = carried fighters)
- Mode selection: Defensive / Offensive / Tolled
- Description of each mode:
  - **Defensive:** Challenge intruders at 1:1 odds (attack/retreat/surrender)
  - **Offensive:** Auto-attack intruders at 2:1 odds
  - **Tolled:** Charge 5 cr/fighter toll or fight at 1:1
- Confirm / Cancel
