# 02-05-PLAN — Ship Systems & StarDock Upgrades

## Objective
Add ship classes and an upgrade system with a StarDock screen where players spend credits to improve their ship. Upgrades tangibly affect gameplay: more cargo = bigger trades, better engines = cheaper jumps, stronger shields/hull = survive combat (Phase 6). This gives players something to work toward beyond "accumulate credits."

## Context
- See `../BRIEF.md` for TW-02 scope
- See `../ROADMAP.md` Phase 5
- Engine types: `packages/engine/src/types.ts`

- CLI screens: `cli/src/screens/`

- Galaxy generator places StarDocks in FedSpace-adjacent sectors

## Tasks

### 1. Add ship types to engine (`packages/engine/src/ships/`)

**ShipClass definitions** — each class has base stats:

```typescript
export interface ShipClass {
  id: string;
  name: string;
  description: string;
  baseStats: ShipStats;
}

export interface ShipStats {
  maxCargo: number;
  maxHull: number;
  maxTurns: number;
  baseTurnCost: number;  // turns per jump (engines can reduce)
  combatBonus: number;   // damage bonus (weapons add)
  dodgeChance: number;   // flee/evade probability (engines add)
}
```

**Starting classes (3 choices):**

| Class | MaxCargo | MaxHull | MaxTurns | TurnCost | Focus |
|-------|----------|---------|----------|----------|-------|
| Merchant | 120 | 100 | 80 | 1 | Cargo capacity |
| Scout | 60 | 80 | 120 | 1 | Speed & turns |
| Interceptor | 70 | 120 | 100 | 1 | Combat durability |

**Upgrade definitions:**

```typescript
export interface Upgrade {
  id: string;
  name: string;
  description: string;
  category: 'engines' | 'shields' | 'weapons' | 'cargo' | 'hull';
  level: number;           // 1-5
  cost: number;
  effect: Partial<ShipStats>;  // additive bonuses
}
```

**Upgrade catalog (5 categories × 3 tiers = 15 upgrades):**

| Category | Tier 1 | Tier 2 | Tier 3 | Effect per tier |
|----------|--------|--------|--------|-----------------|
| Cargo Holds | 2,000 cr | 5,000 cr | 12,000 cr | +30/+40/+50 cargo |
| Engines | 3,000 cr | 8,000 cr | 15,000 cr | -0 turn cost at tier 2 (warp efficiency), +5% dodge |
| Hull Plating | 2,500 cr | 6,000 cr | 12,000 cr | +20/+30/+50 max hull |
| Shields | 4,000 cr | 10,000 cr | 20,000 cr | +15/+25/+40 shield (damage absorb) |
| Weapons | 3,500 cr | 9,000 cr | 18,000 cr | +5/+10/+15 combat damage |

Rules:
- Must buy tiers in order (can't skip to Tier 3)
- Each tier can only be bought once
- Total max investment: ~$100k credits (gives mid-game goal)

**Player ship state (extends existing):**

```typescript
export interface PlayerShip {
  name: string;
  classId: string;
  credits: number;
  currentSector: number;
  cargo: Record<Commodity, number>;
  hull: number;
  turns: number;
  maxTurns: number;
  upgrades: Record<string, number>;  // upgradeId → level owned (0 = not bought)
}
```

## Notes
- **Scope guard:** Combat *damage* is Phase 6. Ship weapons/shields *exist* as purchasable upgrades now but their combat effect is just stored, not used in fights yet.
- **Turn cost:** Currently every jump costs 1 turn. Engine upgrades will reduce this in Phase 6 (e.g., tier 2+ = 0 turns for short jumps). For now, just store the stat.
- **No selling upgrades:** Once bought, permanent. Matches TW2002 behavior.
- **Shield is separate from hull:** Hull = HP. Shield = damage absorption layer. Shields absorb X damage before hull takes hits. Implementation in Phase 6.
